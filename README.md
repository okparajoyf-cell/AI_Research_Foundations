# Day 1: Scalars, Vectors, and Tensors
### Objective
To understand how numerical data is represented in AI/ML frameworks using NumPy.

### Key Concepts Learned
- **Scalars (0D):** Single values, used for weights and biases.
- **Vectors (1D):** Ordered arrays, used for feature sets.
- **Tensors (N-D):** The generalized structure used for deep learning (e.g., images).

### Key Takeaway
Understanding "Shape" is critical for debugging, as it determines how data interacts during matrix multiplication (Broadcasting).
